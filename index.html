🔬 Key Finding: Terminal Boundary Bias

  • No Lost-in-the-Middle. Both models show 100% recall at positions 0.00–0.90 across all tested lengths.
  • Systematic failure at position 1.00. When the needle is placed immediately before the question, recall drops to 0% in 87.5% of cases (7/8 for V4 Pro, 5/6 for Chat).
  • Recovery at 64K+. At 65,536 tokens and above, the boundary bias disappears — KV-cache saturation redistributes attention weights, paradoxically fixing the blind spot.
  • Architectural, not reasoning-related. The non-reasoning Chat model exhibits the identical failure pattern. This is a structural property of the transformer decoder, not a side effect of chain-of-thought tokens.

⚙️ Methodology

  • Haystack: 250 KB neutral text (economics, physics, biology, CS) — no semantic bias
  • Needle: "The secret verification code is MARMALADE-SUNSET-42"
  • Grid: 8–9 context lengths × 7 positions = 56–63 cells per model
  • Temperature: 0.0 — deterministic, reproducible
  • Scoring: Case-insensitive substring match on expected answer
  • Reproduce: pip install httpx && python benchmark.py (see repo)
